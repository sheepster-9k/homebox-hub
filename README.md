# Homebox Hub

Unified Home Assistant integration for Homebox inventory management.

## Features

- Sensors for items, locations, and total value
- Conversation agent for natural-language inventory queries
- Device linking between HA devices and Homebox items
- Home Assistant services for inventory operations
- Sidebar panel for quick access
- Lovelace chat card for inline interaction

## Installation

### HACS (recommended)

1. Open HACS in Home Assistant
2. Go to Integrations > three-dot menu > Custom repositories
3. Add this repository URL with category **Integration**
4. Search for "Homebox Hub" and install

### Manual

Copy the `custom_components/homebox_hub/` directory into your Home Assistant `custom_components/` folder and restart.
